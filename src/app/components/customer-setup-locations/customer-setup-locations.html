Connector Allocation

What is Connector Allocation?

Connector allocation lets you specify how many connectors are assigned to each of your customer's locations. This controls how many concurrent connections your customer's users and resources can support.

Setting a Specific Number

Enter a number in the allocation field to cap how many connectors of that type can be active at this location. This is useful when you want to control resource usage or enforce licensing limits for a specific customer site.

Using the Unlimited Option

Check the Unlimited option to allow an unrestricted number of connectors at this location. The allocation input will be disabled and the connector will not be subject to any cap. This is recommended for locations where usage is unpredictable or where you want to guarantee uninterrupted access regardless of scale.
